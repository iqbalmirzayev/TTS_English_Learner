## Introduction

Learning a new language can be a challenging task, especially when it comes to building vocabulary and improving pronunciation. The English Learning App aims to make the language learning process more engaging and interactive. By leveraging the power of technology, this app allows users to input words, listen to their correct pronunciation, and respond with their own answers in real-time.

## Features

- **Vocabulary Input**: Users can enter words or phrases they want to practice and improve their English vocabulary.
- **Pronunciation Playback**: The app utilizes text-to-speech functionality to provide users with the correct pronunciation of the inputted words.
- **Real-time Answering**: Users can answer the pronunciation by speaking the word themselves, enabling immediate feedback and practice.
- **Modular Structure**: The program is built using Python and incorporates other third-party modules for enhanced functionality and efficiency.

## Usage

1. Launch the English Learning App by running the following command:

   ```bash
   python app.py
   ```

2. Enter the words or phrases you wish to practice.
3. The app will generate the correct pronunciation for the inputted words.
4. Speak the words yourself to practice your pronunciation.
5. The app will provide immediate feedback and indicate whether your pronunciation is correct or not.
6. Continue practicing with new words or phrases as desired.
